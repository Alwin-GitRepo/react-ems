                                       EMS BackEnd Creation using NodeJS and Express
                        ........................................................................
1. Create a new folder named as ems-BackEnd
2. Create package.json file : npm init -y
3. Install Express : npm i express (NodeJS Framework)
4. install cors : npm i cors (The word CORS stands for “Cross-Origin Resource Sharing”. Cross-Origin Resource Sharing is an HTTP-header based mechanism implemented by the browser which allows a server or an API(Application Programming Interface) to indicate any origins (different in terms of protocol, hostname, or port) other than its origin from which the unknown origin gets permission to access and load resources. The cors package available in the npm registry is used to tackle CORS errors in a Node.js application.)
5. Create index.js (main entry point)
6. Create a new folder named as services.inside the folder, create a file called db.js (node + mongodb connection)
7. install mongoose : npm i mongoose (Mongoose is a MongoDB object modeling tool designed to work in an asynchronous environment)
